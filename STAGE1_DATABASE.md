# Stage 1: Database Setup - Step-by-Step

## Sirf Ek Kaam Karna Hai:

### SQL Execute Kro Supabase Mein

```
1. Supabase.com par jao
2. Apna project kholo
3. Left sidebar mein "SQL Editor" click karo
4. create_subscribers_table.sql file open karo (apne project se)
5. Sab code copy karo
6. Supabase SQL editor mein paste karo
7. "Run" button daba do (ya Ctrl+Enter)
8. "Success" message dekhoge
```
